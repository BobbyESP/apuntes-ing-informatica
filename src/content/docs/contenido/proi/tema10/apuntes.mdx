## üìå Concepto de Puntero

Un **puntero** es una variable que almacena la direcci√≥n de memoria de otra variable. En C++, los punteros son fundamentales para manejar memoria din√°mica y optimizar algoritmos que acceden frecuentemente a grandes vol√∫menes de datos.

### Caracter√≠sticas principales:

1. **Direcci√≥n de memoria:** Un puntero almacena la ubicaci√≥n de otra variable en memoria.
2. **Tipos:** Siempre debe declararse el tipo de dato al que apunta el puntero.
3. **Declaraci√≥n:**
   ```cpp
   tipo *nombrePuntero;
   ```
   Ejemplo:
   ```cpp
   int *pEntero;
   float *pReal;
   ```
4. **Operadores asociados:**
   - `&`: Devuelve la direcci√≥n de memoria de una variable.
   - `*`: Accede al contenido almacenado en la direcci√≥n apuntada por el puntero.

#### Ejemplo:

```cpp frame="terminal"
#include <iostream>
using namespace std;

int main() {
    int valor = 42;
    int *p = &valor; // El puntero almacena la direcci√≥n de 'valor'

    cout << "Direcci√≥n de valor: " << p << endl;
    cout << "Contenido apuntado por p: " << *p << endl;

    *p = 84; // Modifica 'valor' usando el puntero
    cout << "Nuevo valor de valor: " << valor << endl;

    return 0;
}
```

Salida:

```
Direcci√≥n de valor: 0x7ffee59a4a8
Contenido apuntado por p: 42
Nuevo valor de valor: 84
```

## üõ†Ô∏è Asignaci√≥n y Liberaci√≥n de Memoria

En C++, la memoria puede reservarse din√°micamente durante la ejecuci√≥n del programa. Esto es crucial cuando no se conoce el tama√±o de los datos en tiempo de compilaci√≥n.
La gesti√≥n de esta memoria din√°mica se realiza en el **mont√≥n (heap)**, un √°rea de memoria donde se alojan las variables din√°micas, cuya memoria se reserva
y libera en tiempo de ejecuci√≥n.

### 1.  `new`: Reserva memoria
La palabra clave `new` se usa para solicitar un bloque de memoria del tama√±o del tipo de dato especificado. Devuelve un puntero a la primera posici√≥n de la memoria asignada.

```cpp

*p = 10;          // Asigna el valor 10 a esa memoria
```

Para arrays:

```cpp
int *arr = new int[5]; // Reserva memoria para un array de 5 enteros
```

### 2. `delete`: Libera memoria

Es crucial liberar la memoria asignada din√°micamente con `delete` cuando ya no se necesita. Esto previene fugas de memoria, un problema com√∫n cuando la memoria reservada no se libera.

```cpp
delete p;       // Libera la memoria de un entero
delete[] arr;   // Libera la memoria de un array
```

#### Ejemplo completo

```cpp frame="terminal"
#include <iostream>
using namespace std;

int main() {
    int *p = new int;
    if (p == nullptr) { //nullptr es un puntero nulo
        cerr << "Error al asignar memoria." << endl;
        return 1;
    }
    *p = 20;
    cout << "Valor almacenado: " << *p << endl;

    delete p; // Libera la memoria
    return 0;
}
```

## ‚ûï Operaciones con Punteros

### Direccionamiento e Indireccionamiento

Como ya dijimos, el operador `&` obtiene la direcci√≥n de memoria de una variable, mientras que `*` accede al contenido en esa direcci√≥n.
Con estes operadores, podemos manejar las variables a nuestro gusto.

#### Ejemplo


### Comparaci√≥n de Punteros

Los punteros pueden compararse con operadores como `==` o `!=` para verificar si apuntan a la misma direcci√≥n.

#### Ejemplo

```cpp

```

### Aritm√©tica de Punteros

Se pueden realizar operaciones como suma, resta y comparaci√≥n sobre punteros. Estas operaciones son √∫tiles al trabajar con arrays.

#### Ejemplo

## üîÑ Formas de Acceso a Variables y Arrays con Punteros

Los punteros permiten acceder y manipular variables o arrays de diferentes maneras. A continuaci√≥n, exploraremos las t√©cnicas m√°s comunes:

### 1. Acceso Directo

Usando el operador de indireccionamiento (`*`) para acceder al valor almacenado en la direcci√≥n apuntada por el puntero.

#### Ejemplo

```cpp

cout << "Valor: " << *p << endl; // Acceso directo al contenido
```

### 2. Recorrido de Arrays

Un puntero puede moverse dentro de un array usando aritm√©tica de punteros.

#### Ejemplo

```cpp

    cout << "Elemento " << i << ": " << *(p + i) << endl;
}
```

### 3. Iteraci√≥n con Incremento

El puntero puede ser incrementado para recorrer secuencialmente un array.

#### Ejemplo

```cpp

int *p = arr;

while (p < arr + 3) {
    cout << *p << " ";
    p++; // Incrementa el puntero
}
```

### 4. Manipulaci√≥n de Subarrays

Un puntero puede apuntar a una secci√≥n espec√≠fica de un array y operar solo con esa parte.

#### Ejemplo:

```cpp

    cout << "Elemento: " << *(subArray + i) << endl;
}
```

## üèóÔ∏è Arrays Din√°micos

Un array din√°mico es aquel cuyo tama√±o se determina en tiempo de ejecuci√≥n. Son √∫tiles para trabajar con estructuras de datos que var√≠an en tama√±o.

#### Ejemplo

```cpp frame="terminal"
    int n;
    cout << "Introduce el tama√±o del array: ";

    return 0;

```
